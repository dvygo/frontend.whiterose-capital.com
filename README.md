# frontend.whiterose-capital.com

Next.js app with **Tailwind CSS** and **GSAP**.

## Setup

```bash
npm install
```

## Run

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000). The home page uses GSAP for the hero animation.

## Stack

- **Next.js** 14 (App Router)
- **Tailwind CSS** 3
- **GSAP** 3
- **TypeScript**
